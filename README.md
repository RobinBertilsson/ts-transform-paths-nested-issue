# Monto - React Common UI

`@monto/react-common-ui` is a library that contains core elements & components.

## Installation

Using the package manager NPM.

```bash
npm install @monto/react-common-ui
```

## Usage

```tsx
import { Button } from '@monto/react-common-ui'

const Example = () => (
  <Button>
    Hello World
  </Button>
)
```
